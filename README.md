# Multithreading in C++

## Overview

This project demonstrates the implementation of multithreading concepts using C++. It was developed as part of learning Operating Systems and Concurrent Programming.

## Features

- Thread creation
- Parallel execution
- Thread synchronization
- Clean and modular C++ code

## Technologies Used

- C++
- std::thread
- GCC Compiler
- Linux

## How to Run

```bash
g++ main.cpp -pthread -o app
./app
